# Cloudflare Tunnel

## Helpers

- `create-secret.sh` is a helper to create the cloudflare tunnel secret required by the docker-compose file.
- `token-to-config.sh` and `config-to-token.sh` are helpers to convert the token to a config file and back.
  Inspiration <https://github.com/cloudflare/cloudflared/issues/645#issuecomment-2735272361>.
